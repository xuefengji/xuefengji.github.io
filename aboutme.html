致力于前后端开发技术
一个90后程序员，终身学习者。
  关于文渊

我是文渊，网名闻鸢同学。

角色：前端空城师 —— 一个 90 后程序员，终身学习者。

工作：保持对技术的热情，保持谦虚之心，保持学习的动力。—— 只要投入时间去学，没有学不会的东西。

技能：主职业是前端，后端、运维、大数据的技术栈也有所涉及。

副技能：抽空在学习设计、产品的知识，如果逮住合适的机会，也会培养运营和营销的技能。

娱乐：爱阅读，但不包括文学名著；二次元爱好者，但不是 C 圈的宅男；喜欢 MMORPG 游戏，但更想把时间花在好好奋斗上。

心态：爱生活、爱阅读、爱编程、爱学习、爱拼搏、爱大自然。

原则：常怀感恩的心。

社交：一个人走得快，一群人走得远。

理财：人生就像滚雪球。最重要之事是发现够湿的雪和长长的山坡。

心路历程：

小时候最喜欢学校的电脑课，经常在课上偷偷倒腾 QQ 空间，没有钱买装饰，就学别人植入各种“代码”来装扮。

中学时候看了《黑客帝国》，迷恋上了 Hacker，想象对着满屏幕英文，劈里啪啦一顿乱敲帅气的样子。

上大学前认知不够开明，不知道计算机专业出来能做什么，于是填志愿去了自动化专业。但对软件的兴趣甚于硬件，因此利用课余时间自学 Java 技术栈。

出来社会后，运气好通过了面试，转型到了 IT 行业。干了一年运维和辅助开发的工作后，开始主做前端。前端能及时给出视觉反馈的特点，成了我热衷这个岗位的最大原因。

主力前端并不影响我持续学习的欲望，不论是程序员的各项软硬技能，还是其他领域的知识，我都会在空余时间不断接触和尝试。

有着想去大厂的梦想，不过条条大路通罗马，大厂是一条路但不是唯一的路，因而充实好自己最重要，有机会就努力抓住机会，没有机会就自己创造机会。

这就是我，一个普通的程序员，一个不甘平庸的 90 后，奥力给！
