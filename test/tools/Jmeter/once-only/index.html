# 什么是 Once Only 控制器

  • 每个线程内，该控制器下的内容只会被执行一次

  • 无论循环多少次，都只执行一遍

应用场景：

  • 对于需要登录的测试，可以将登录请求放置在此控制器中，因为每个线程只需要登录一次即可建立会话
