# Switch Controller

作用：控制该控制器下的取样器执行

应用场景：可以用在模拟多线程同时操作不同请求的测试场景

# 界面介绍

Switch Value：

  • 数字
    • 0 表示运行 Switch 控制器里的第一个请求，1 表示运行第二个请求以此类推
    • 数字超过取值范围了，就默认是 0
    • 不填：默认执行数字 0 的请求，即顺位第一个请求
  • 字符：输入 Switch 控制器里面请求的名称，匹配与请求中的名称相同则执行该请求
  • 变量
