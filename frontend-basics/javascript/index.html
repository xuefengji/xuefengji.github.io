# 行为层 JavaScript

在 Web 前端页面中，

  • HTML 属于结构层，负责描绘出内容的结构；
  • CSS 属于表现层，负责“如何显示有关内容”；
  • JavaScript 属于行为层，负责“内容应如何对事件做出反应”。

关于 JavaScript 的知识点，主要参考：

  • 《JavaScript高级程序设计（第3版）》
  • 《你不知道的JavaScript（上卷）》
  • 《你不知道的JavaScript（中卷）》
  • 《ES6标准入门（第3版）》
  • MDN 中关于 JavaScript 的内容
