# 结构层 HTML

在 Web 前端页面中，

  • HTML 属于结构层，负责描绘出内容的结构；
  • CSS 属于表现层，负责“如何显示有关内容”；
  • JavaScript 属于行为层，负责“内容应如何对事件做出反应”。
