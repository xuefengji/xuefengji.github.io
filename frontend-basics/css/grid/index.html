# Grid 布局

Grid 网格布局虽然强大，但由于其兼容性比 Flex 更差一点，目前并未得到全面应用。这主要取决于现代化浏览器的普及率（有些单位还是会更多地使用 IE）。

但作为一种全新的布局方式，随着现代化浏览器的逐渐普及，再加上不可能总是做面向 IE 客户的产品，Grid 网格布局未来也许会是一种趋势。

因此这一章节先放两篇比较好的Grid 网格布局文章，等以后用到了再系统整理。
