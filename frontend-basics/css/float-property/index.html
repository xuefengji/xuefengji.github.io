# 浮动（float）

# 1. 浮动的特点

浮动盒子可以向左或向右移动，直到其外边沿接触包含块的外边沿，或接触另一个浮动盒子的外边沿。

浮动盒子会脱离常规文档流，后面的元素盒子会当它不存在，进行正常的排列。但后面元素盒子中的文本内容，在排布时会避开该浮动盒子，形成文本环绕的效果。

<html>
  <div class="float-demo-1">
    <div class="box-1"></div>
    <div class="box-2"></div>
    <div class="box-3">文本内容文本内容文本内容文本内容文本内容文本内容文本内容文本内容文本内容文本内容文本内容文本内容</div>
  </div>
</html>

<style>
.float-demo-1 {
  width: 300px;
  height: 200px;
  background-color: gray;
}
.float-demo-1 .box-1 {
  float: left;
  width: 100px;
  height: 100px;
  background-color: red;
}
.float-demo-1 .box-2 {
  float: left;
  width: 100px;
  height: 100px;
  background-color: green;
}
</style>

# 2. 清除浮动

要阻止行盒子里的文本环绕在浮动盒子外面，需要给包含行盒子的元素应用 clear 属性。clear 属性的值有 left、right、both 和 none，用于指定盒子的哪一侧不应该紧挨着浮动盒子。




























 



<html>
  <div class="float-demo-2">
    <div class="box-1"></div>
    <div class="box-2"></div>
    <div class="box-3">文本内容文本内容文本内容文本内容文本内容文本内容文本内容文本内容文本内容文本内容文本内容文本内容</div>
  </div>
</html>

<style>
.float-demo-2 {
  width: 300px;
  height: 200px;
  background-color: gray;
}
.float-demo-2 .box-1 {
  float: left;
  width: 100px;
  height: 100px;
  background-color: red;
}
.float-demo-2 .box-2 {
  float: left;
  width: 100px;
  height: 100px;
  background-color: green;
}
.float-demo-2 .box-3 {
  clear: both;
}
</style>
