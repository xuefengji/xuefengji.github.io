CSS

# CSS

在 Web 前端页面中，

  • HTML 属于结构层，负责描绘出内容的结构；
  • CSS 属于表现层，负责「如何显示有关内容」；
  • JavaScript 属于行为层，负责「内容应如何对事件做出反应」。

关于 CSS 的知识点，主要参考：

  • 《精通CSS 高级Web标准解决方案（第3版）》
  • 《深入解析CSS》
  • W3school 的 HTML/CSS 部分
  • MDN 中关于 CSS 的内容
